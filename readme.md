# Personal Budget Manager

A simple Java Swing application for managing personal income and expenses.

## Features
- Add income
- Add expenses
- Set monthly budget
- View summary
- Delete transactions
- Search by date or category
- Expense pie chart using JFreeChart

## Technologies
Java  
Swing GUI  
File Handling  
JFreeChart
